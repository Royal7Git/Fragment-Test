Fragment Example :

This repositary holds the files that are used to demonstrate the example of the Fragment in an Android Application.
It contains the 3 JAVA files which are used to code the backend and 3 XML files used to create the Layout (GUI) of application.

You can copy-paste and replace these files directly into your project.
